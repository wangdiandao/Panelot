Panelot Privacy Policy / 隐私政策

Effective date / 生效日期：2026-07-10

中文

Panelot 是一款本地优先、由用户自带模型密钥的浏览器 Agent 扩展。我们不运营遥测、广告、账户或云同步服务。

会话、配置、附件和加密后的凭据存储在浏览器本地。仅在用户发起模型请求、MCP 请求或网页操作时，相关数据才会发送给用户选择的服务或网站。

默认数据导出不包含任何密钥、Bearer token、OAuth refresh token 或敏感 Header。用户明确选择的含秘密备份会先用口令加密。

扩展按需请求网站访问权限；用户可以在浏览器扩展设置中随时撤销。Panelot 不出售或共享个人数据。

问题与安全报告：GitHub Security。

English

Panelot is a local-first browser agent extension that uses model credentials supplied by the user. We operate no telemetry, advertising, account, or cloud-sync service.

Chats, settings, attachments, and encrypted credentials remain in browser-local storage. Relevant data is sent only when the user invokes a model, MCP server, or website operation, and only to the destination selected by the user.

Default exports exclude API keys, bearer tokens, OAuth refresh tokens, and sensitive headers. Secret-bearing backups are encrypted with a user passphrase.

Website access is requested only when needed and can be revoked from browser extension settings. Panelot does not sell or share personal data.

Questions and security reports: GitHub Security.
