# 用户指南

Panelot 是运行在 Chrome 和 Edge 中的本地优先 AI Agent 扩展。你可以连接自己的模型服务，在侧边栏或全页对话中提问、引用网页与文件，并让模型在你允许的范围内操作浏览器。

Panelot 不提供账户、云同步或自有模型服务。开始使用前，你需要准备一个可用的 OpenAI 兼容接口或 Anthropic 接口；调用费用、数据处理和可用模型由对应服务决定。

## 使用前需要知道

- Panelot 当前面向 Chrome 116 或更高版本，以及基于同一扩展平台的 Edge；不提供 Firefox 或 Safari 构建。
- 页面正文不会仅因为侧边栏打开就自动发送给模型。你需要明确引用页面，或让 Panelot 在任务中读取页面。
- 网站、模型服务、MCP 服务器、Skill 和 Plugin 均可能包含不可信内容。执行前应核对审批卡片中的目标、参数和将要发送的数据。
- Panelot 可以降低误操作风险，但不能替代你对付款、账户、隐私和数据外发的判断。凭据、验证码、支付和真人验证应由你接管完成。
- “OpenAI 兼容”只表示使用相应请求格式，不代表任意端点都已通过 Panelot 的真实兼容测试。添加连接后应先运行验证，再发起一次实际对话。
