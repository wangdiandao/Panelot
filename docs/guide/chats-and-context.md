# 对话与上下文

## 侧边栏与全页对话

点击工具栏中的 Panelot 图标或按 `Alt+P` 打开侧边栏。侧边栏适合围绕当前页面快速提问；点击顶部的展开按钮或按 `Ctrl/Cmd+E` 可在全页对话中打开同一会话。

全页对话提供会话列表、搜索、置顶、重命名和删除。删除会话后无法恢复。新会话草稿不会立即写入历史记录，只有成功提交第一条消息后才会成为正式会话。

侧边栏会记住最后选择的有效会话。关闭并重新打开浏览器或侧边栏后，会优先恢复该会话；如果它已经被删除或不可用，则回退到最近的有效会话。

## 发送、插话与排队

- `Enter`：发送；`Shift+Enter`：换行。
- 任务运行中输入并按 `Enter`：在下一次模型调用前插话。当前步骤不能插话时，会自动加入队列。
- `Shift+Alt+Enter`：明确把消息加入队列，等待当前轮结束。
- `Esc`：停止当前轮。
- 输入框为空时按 `↑`：召回上一条输入。

停止或关闭界面不一定能撤销已经发给网站或远端服务的操作。对于结果状态不明的写操作，Panelot 不会自动重复执行，而会要求你确认结果。

## 引用网页和其它上下文

Panelot 不会自动把所有打开页面加入模型上下文。可以使用以下方式明确选择内容：

- 输入 `@`：选择一个打开的普通网页，或选择已连接 MCP 服务器提供的 Resource。
- 点击输入框中的 `+`：上传文件、引用打开的标签页，或选择一个已启用 Skill。
- 侧边栏的当前页面提示：点击“附着到对话”后，页面正文才成为上下文胶囊。
- 输入 <code v-pre>{{</code>：插入发送时求值的变量，包括当前页 URL、标题、选中文本、剪贴板和当前日期。

引用页面时，Panelot 会提取主要可读文本并限制体积；过长内容会截断。网页内的导航、脚本、页脚和部分嵌入内容可能不会进入这份轻量正文。若任务需要更完整的页面结构，可以直接让 Panelot 读取页面，模型会根据任务调用浏览器读取能力。

本轮浏览器操作默认以当前活动网页为目标，但这不代表页面正文已经发送给模型。引用其它标签页也不会切换浏览器前台标签页。

## 文件、粘贴内容和截图

单个上传文件不能超过 **8 MB**。新会话要先发送一条消息，成为正式会话后才能持久化文件；否则界面会提示先发送消息。

上传后，模型只能看到文件名、类型、大小和附件编号，浏览器上传工具可以据此选择文件。Panelot 不会自动解析文件正文，也不会把文件字节作为普通模型附件发送。若要分析其中的文字，请把相关文本粘贴到上下文中。

在输入框粘贴超过 2000 个字符的文本时，Panelot 会把它变成单独的上下文胶囊，避免输入框过长。按住 `Shift` 粘贴可以保留为普通输入文本。

网页截图和页面摘录也可能作为附件保存在当前会话。附件可在“设置 → 附件”中查看和删除；删除后，引用它的历史消息仍然存在，但对应文件会显示为不可用。数据 JSON 导出不包含这些附件文件，详见[数据与隐私](./data-and-privacy.md)。

## Skills 与斜杠命令

输入 `/` 可以选择：

- 已启用 Skill 注册的命令；
- 已连接 MCP 服务器提供的 Prompt；
- 当前界面提供的内置命令。

有参数的命令会先显示表单。Skill 和 MCP Prompt 会把生成的指令内容附加到当前消息，它们不会绕过浏览器权限或替你授权外部操作。

## 编辑、重新生成和分支

编辑已发送的用户消息或重新生成助手回复会创建一个新分支，不会覆盖原分支。消息旁的分支切换器可在不同结果之间切换。任务运行中会暂时禁用这些分支操作。

原对话会保留；切换分支后，后续模型上下文只沿当前分支继续。

## 常用快捷键

| 快捷键             | 作用                     |
| ------------------ | ------------------------ |
| `Alt+P`            | 打开或关闭侧边栏         |
| `Ctrl/Cmd+K`       | 打开命令面板             |
| `Ctrl/Cmd+N`       | 新建会话                 |
| `Ctrl/Cmd+,`       | 在全页对话中打开设置     |
| `Ctrl/Cmd+E`       | 在侧边栏与全页对话间切换 |
| `Ctrl/Cmd+Shift+S` | 收起或展开全页会话列表   |
| `?`                | 打开快捷键帮助           |
| `Ctrl/Cmd+Shift+C` | 复制最后一条回复         |

审批卡片还有固定的 `Y / S / A / N` 快捷键，详见[浏览器操作与权限](./browser-and-permissions.md)。
