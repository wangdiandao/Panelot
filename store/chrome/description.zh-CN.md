# Panelot — 浏览器原生 AI Agent

Panelot 将 Chrome 变成本地优先的 AI 工作区。连接你自己的 OpenAI 兼容或 Anthropic 兼容模型，在明确的安全边界内读取页面并执行网页操作。

核心能力：

- 本地会话树、附件、Skills、模型预设与 Plugin
- 仅在具体操作需要时申请网站访问权限
- 读取、点击、输入、导航、截图和上传用户选择的文件
- 通过 Streamable HTTP 接入 MCP 工具、Prompt 与 Resource
- 操作审批、敏感站点保护与可恢复的任务 Run
- 本机加密凭据与口令加密的含秘密备份
- 无账户、无云同步、无广告、无遥测

Panelot 只会把相关内容发送到用户选择的模型端点、MCP 服务或网站。Provider 与 MCP 凭据加密保存在浏览器本地。
