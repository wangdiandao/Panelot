# Panelot — Chrome AI Agent

连接你自己的 OpenAI 兼容或 Anthropic 端点，向 Panelot 询问网页内容，并让它按当前权限模式和规则完成浏览器任务。

Panelot 支持：

- 本地会话树、附件、Skills、模型预设与 Plugin
- 仅在具体操作需要时申请网站访问权限
- 读取、点击、输入、导航、截图和上传用户选择的文件
- 通过 Streamable HTTP 接入 MCP 工具、Prompt 与 Resource
- 三种权限模式、已保存规则、敏感站点保护与可恢复的任务 Run
- 本机加密凭据与口令加密的含秘密备份
- 无账户、无云同步、无广告、无遥测

Panelot 会把用户选入的内容发送到已配置的模型端点或 MCP 服务器；浏览器操作会与正在处理的网站通信。Provider 与 MCP 凭据加密保存在扩展本地。
